# Project Aegis: ISO 27001 Implementation Sandbox

This repository contains the artifacts from a simulated GRC audit of a mid-sized tech firm.

## Contents
- **Gap Analysis Tool (Excel):** Automated scoring and risk heat map.
- **Policy Framework:** Sample Human Resource Security Policy (A.6).
- **Executive Board Deck:** Summary of risk exposure and remediation roadmap.
