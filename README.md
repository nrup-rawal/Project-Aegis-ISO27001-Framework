### Executive Summary
**Project Aegis** is a simulated Governance, Risk, and Compliance (GRC) implementation for a mid-sized technology firm. The objective was to audit the organization against **ISO/IEC 27001:2022 Annex A** controls, quantify financial risk, and design a remediation roadmap.

### üîç Methodology

This project follows the **PDCA (Plan-Do-Check-Act)** cycle:

1.  **Gap Analysis:** Audited 20 critical controls across People, Physical, and Technological domains.
    * *Result:* 55% Compliance / 45% Critical Gaps.
2.  **Risk Assessment:** Utilized **ISO 27005** methodology ($Impact \times Likelihood$) to prioritize remediation.
    * *Critical Find:* Lack of MFA (Score 25) posed a higher financial risk than outdated documentation (Score 8).
3.  **Remediation:** Designed a Q1 Roadmap to implement Microsoft Entra ID (MFA) and background verification workflows.
